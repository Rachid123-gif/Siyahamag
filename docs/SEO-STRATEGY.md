## 3. STRATEGIE TECHNIQUE SEO (a implementer)

### 3.1 Fondations critiques

| Action | Priorite | Statut | Impact |
|--------|----------|--------|--------|
| Sitemap XML dynamique | URGENTE | ❌ Manquant | Google crawle toutes les pages |
| robots.txt | URGENTE | ❌ Manquant | Controle du crawl |
| Canonical URLs sur toutes les pages | HAUTE | ❌ Manquant | Evite le contenu duplique |
| JSON-LD sur TOUTES les pages | HAUTE | ⚠️ 2 pages seulement | Rich snippets Google |
| Open Graph complet | MOYENNE | ✅ Fait (47 pages) | Partage social |
| Meta description unique par page | HAUTE | ✅ Fait | CTR dans les SERPs |
| Vitesse de chargement < 2s | HAUTE | ✅ Bon (Next.js SSR) | Core Web Vitals |
| Mobile-first responsive | HAUTE | ✅ Fait | Mobile-first indexing |
| HTTPS | HAUTE | ⏳ En cours (Netlify) | Facteur de ranking |
| Hreflang fr-MA | MOYENNE | ❌ Manquant | Ciblage geographique |
| Schema.org WebSite | HAUTE | ❌ Manquant | Sitelinks Google |

### 3.2 Schemas JSON-LD a implementer

| Page | Schema | Benefice |
|------|--------|----------|
| / (homepage) | WebSite + SearchAction | Barre de recherche dans Google |
| /actualites/* | Article + BreadcrumbList | Rich snippets articles |
| /emplois/* | JobPosting | Rich snippets emploi (logo, salaire, lieu) |
| /investissement/* | RealEstateListing | Rich snippets immobilier |
| /statistiques | Dataset | Donnees structurees |
| /entreprise/* | Organization | Knowledge panel |
| Toutes les pages | BreadcrumbList | Navigation dans les SERPs |

### 3.3 Performance (Core Web Vitals)

| Metrique | Cible | Levier |
|----------|-------|--------|
| LCP (Largest Contentful Paint) | < 2.5s | Images optimisees, lazy loading |
| FID (First Input Delay) | < 100ms | Minimiser le JS client |
| CLS (Cumulative Layout Shift) | < 0.1 | Dimensions images explicites |
| TTFB (Time to First Byte) | < 600ms | SSR + CDN Netlify |

## 4. STRATEGIE DE CONTENU

### 4.1 Calendrier editorial (publier 3 articles/semaine minimum)

| Jour | Type de contenu | Categorie |
|------|----------------|-----------|
| Lundi | Actualite du secteur | Gouvernement / Invest |
| Mercredi | Analyse / Guide pratique | Thematiques |
| Vendredi | Chiffres / Tendances | Marches / Statistiques |

### 4.2 Types de contenu a forte valeur SEO

**A. Articles piliers (3000+ mots) — 1/mois**
Objectif : Devenir LA reference sur un sujet. Exemples :
- "Guide complet de l'emploi dans le tourisme au Maroc en 2026"
- "Investir dans un riad au Maroc : tout ce qu'il faut savoir"
- "Statistiques du tourisme marocain : bilan complet 2025"
- "Les metiers du tourisme au Maroc : salaires, formations, perspectives"
- "Coupe du Monde 2030 : impact sur le tourisme marocain"

**B. Articles evergreen (1500+ mots) — 2/mois**
Contenu intemporel, mis a jour regulierement :
- "Top 10 des villes touristiques du Maroc"
- "Comment ouvrir un hotel au Maroc : guide etape par etape"
- "Les formations en hotellerie et tourisme au Maroc"
- "Salaire moyen dans l'hotellerie au Maroc par poste"

**C. Articles d'actualite (500-800 mots) — 3/semaine**
Reactif, chaud, partage social :
- Ouvertures d'hotels
- Decisions gouvernementales
- Chiffres mensuels
- Evenements

**D. Pages de donnees (statistiques) — mises a jour trimestrielles**
- Nombre de touristes par mois/annee
- Recettes par region
- Taux d'occupation par ville
- Comparaisons internationales

### 4.3 Strategie de maillage interne

Chaque article doit contenir :
- 2-3 liens vers d'autres articles (sujets lies)
- 1 lien vers la page emplois (si pertinent)
- 1 lien vers la page statistiques (si chiffres mentionnes)
- 1 lien vers la page investissement (si opportunite mentionnee)

Schema de maillage :
```
Homepage
  ├── Actualites (hub) ──→ Articles individuels
  │     └──→ Emplois (CTA)
  │     └──→ Statistiques (donnees)
  ├── Emplois (hub) ──→ Offres individuelles
  │     └──→ Entreprises (profils)
  │     └──→ Articles lies (contexte)
  ├── Statistiques ──→ Articles d'analyse
  │     └──→ Investissement (opportunites)
  └── Investissement (hub) ──→ Opportunites
        └──→ Articles lies
```
